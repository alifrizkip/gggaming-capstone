# GGGamingCapstone iOS

My Final Submission for [**_IDCamp 2021_**](https://idcamp.indosatooredoo.com/) **_X_** [**_Dicoding iOS Developer path_**](https://www.dicoding.com/learningpaths/9), Expert class.

Course: [Menjadi iOS Developer Expert](https://www.dicoding.com/academies/209).

What I learned from this submission:
- iOS App Development with SwiftUI.
- Clean code and clean architecture with VIPER (View, Interactor, Presenter, Entity, and Router) pattern.
- Modularization with generic protocol approach.
- Publish and create module ([GGGamingCore](https://github.com/alifrizkip/GGGamingCore)) with Swift Package Manager.
- Dependency Injection with [Swinject](https://github.com/Swinject/Swinject).
- Reactive Programming with [Combine](https://developer.apple.com/documentation/combine).
- Localization. English and Bahasa Indonesia.
- Unit test with [XCTest](https://developer.apple.com/documentation/xctest).
- Linter with [SwiftLint](https://github.com/realm/SwiftLint).
- Continuous Integration with [Github Actions](https://github.com/alifrizkip/gggaming-capstone/actions). Steps: lint, test, build.
